# Omihirofumi Tap

## How do I install these formulae?

`brew install omihirofumi/tap/<formula>`

Or `brew tap omihirofumi/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "omihirofumi/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
